## 📂 Sección: Conceptos Avanzados del Sistema de Ficheros (Capítulo 6)

Esta sección está dedicada a las prácticas realizadas en el curso de **Santiago Hernández**. Se enfoca en entender cómo Linux gestiona los datos internamente, más allá de la superficie de las carpetas.

### 🛠️ Conceptos Practicados en esta Sección
* **Gestión de Inodos:** Identificación técnica de archivos mediante su índice único usando `ls -li`.
* **Tipos de Enlaces:**
    * **Hard Links:** Creación de enlaces físicos que comparten el mismo inodo.
    * **Soft Links (Simbólicos):** Creación de accesos directos y gestión de punteros a rutas.
* **Estructura del Sistema:** Navegación y comprensión de la jerarquía de directorios de Linux (FHS).
* **Shell Avanzada:** Optimización del uso de la terminal y gestión del historial de comandos.

### 💻 Comandos Clave Utilizados
| Comando | Función |
| :--- | :--- |
| `ls -li` | Visualizar el número de inodo. |
| `ln` | Crear un Hard Link. |
| `ln -s` | Crear un Soft Link. |
| `readlink` | Verificar el destino de un enlace. |
